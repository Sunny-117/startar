# Starter

基于 N 的模板，N=?, 提供 select 的能力

- Vite
- Esbuild
- SWC
- rollup
- Rspack(Rsbuild)
- rolldown

插件开发，推荐使用 unplugin-starter，同时兼容 vite,rollup,esbuild,webpack,rspack...

## Install

```shell
npm i -g startar
startar
```
![](./assets/usage.png)

## dev

```bash
pnpm i
pnpm dev
npm link
startar
```
